Trackable pages can be created to follow how traffic is reacting so you know immediately what needs to be improved.. This will show you which pages are working and which need help. By knowing information like this it can help you gain more sales by applying more emphasis on what is broken so you can fix your weakest links! 

== Description ==

This plugin will allow you find out how you compare to your competitors. With years of experience and analytics to back it, you will be on your way to making better decisions.

*NEW: Manage your Juiced Metrics*

== Installation ==

1. Upload the `JuicedMetrics` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A new top-level menu called 'Juiced Metrics' will appear in your administration menu.
